# Proyecto-de-Software---Trabajo-Final-Cristaldo-Blazco-
En este repositorio se subiran todos los archivos referidos a la materia de Proyecto de Software - Trabajo Final de la carrera de Analista en Sistemas de Computacion de la Facultad de Ciencias Exactas, Quimicas y Naturales-UNaM. En el cual se llevara a cabo el dearrollo del proyecto de software(Tesina) que lleva como nombre Easy-Rent.

## Idea General del Proyecto

Se desea llevar a cabo el desarrollo de una **plataforma web** que permita mejorar el proceso de alquiler de una propiedad, ofreciendo una administracion mas dinamica y comoda a los **Propietarios** para la publicacion de sus propiedades, asi como tambien a los **Inquilinos** para la busqueda de las mismas.
